Projeto Java JPA Wildfly

Temos aqui um CRUD Java Web com Wildfly Primafaces. O termo CRUD é amplamente utilizado na programação para descrever as quatro operações básicas que podem ser feitas em dados persistentes: Create (Criar), Read (Ler), Update (Atualizar) e Delete (Excluir). Quando implementamos essas operações em um projeto Java Web utilizando tecnologias como JSF, PrimeFaces, JPA e EJBs, é comum referir-se a isso como um CRUD Java Web. Isso descreve de forma sucinta a funcionalidade principal do seu sistema: a capacidade de criar, ler, atualizar e excluir registros de um banco de dados por meio de uma interface web..
